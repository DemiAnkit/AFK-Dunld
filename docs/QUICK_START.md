# Quick Start Guide

Welcome to AFK-Dunld! This guide will help you get started in minutes.

## üì¶ Installation

### Windows
1. Download `AFK-Dunld-Setup.msi` from [releases](https://github.com/yourusername/afk-dunld/releases)
2. Double-click to install
3. Follow the installation wizard
4. Launch from Start Menu

### macOS
1. Download `AFK-Dunld.dmg`
2. Open the DMG file
3. Drag AFK-Dunld to Applications
4. Launch from Applications
5. If Gatekeeper blocks it, right-click ‚Üí Open

### Linux
```bash
# Debian/Ubuntu
wget https://github.com/yourusername/afk-dunld/releases/download/v0.1.0/afk-dunld_0.1.0_amd64.deb
sudo dpkg -i afk-dunld_0.1.0_amd64.deb

./afk-dunld_0.1.0_amd64.AppImage
```

## üöÄ First Download

### Method 1: Simple Download
1. Click **"+ Add Download"** button (or press `Ctrl+N`)
2. Paste your URL
3. Click **"Add"**
4. Watch it download!

### Method 2: YouTube Video
1. Click **"+ Add Download"**
2. Paste YouTube URL
3. Select **Video** or **Audio Only**
4. Choose quality (4K, 1080p, 720p, etc.)
5. Click **"Add"**

### Method 3: Playlist
1. Paste a YouTube playlist URL
2. Playlist dialog opens automatically
3. Select format and quality
4. Click **"Download Playlist"**
5. All videos download!

## üîß Troubleshooting

### Download Won't Start
1. Check internet connection
2. Verify URL is valid
3. Check disk space
4. Try pausing and resuming

### YouTube Download Fails
1. Update yt-dlp in settings
2. Check if video is available
3. Try different quality
4. Check for geo-restrictions

### App Won't Launch
1. Check system requirements
2. Reinstall the application
3. Check antivirus isn't blocking
4. View logs for errors
